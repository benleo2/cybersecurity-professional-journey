# SIEM Basics

## What SIEM Is
SIEM centralizes logs and correlates events to generate security alerts.

## Why SIEM Matters
SIEM enables faster detection and investigation of attacks across systems.

## Log Sources
- Windows logs
- Linux logs
- Web server logs

## Alert Logic
Alerts are generated based on correlation rules and thresholds.

## SOC Insight
SIEM supports analysts but does not replace human analysis.
