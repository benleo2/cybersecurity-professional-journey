# Cookies & Sessions

## What it is
Cookies store session identifiers used to maintain user authentication.

## Why it matters in security
Stolen or misconfigured cookies lead to session hijacking.

## Concepts Learned
- Cookie-based authentication
- Session management
- Security flags

## Hands-on Practice
- Inspected cookies via browser dev tools
- Observed security flags

## Security Insight
Authentication relies on secure session handling.
