# HTTP Basics

## What it is
HTTP is the protocol used for communication between browsers and web servers.

## Why it matters in security
All web attacks occur through manipulated HTTP requests and responses.

## Concepts Learned
- HTTP requests & responses
- Methods (GET, POST)
- Status codes

## Hands-on Practice
- Used curl to inspect HTTP traffic
- Observed headers and status codes
- Analyzed browser network requests

## Security Insight
Understanding HTTP is essential before learning web vulnerabilities.
