# Firewall Basics (UFW)

## What it is
A firewall controls network traffic based on defined rules.

## Why it matters in security
Firewalls reduce exposure and prevent unauthorized access.

## Commands Used
- ufw status
- ufw enable
- ufw allow
- ufw deny

## Hands-on Practice
- Enabled firewall
- Allowed required services
- Denied unnecessary ports
- Reviewed rules

## Defender Perspective
Use default deny and allow only required traffic.
