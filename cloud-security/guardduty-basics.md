## What GuardDuty Is
A managed AWS service that detects malicious activity using logs and threat intelligence.

## Detection Sources
- CloudTrail
- VPC Flow Logs
- DNS Logs

## Security Use Cases
- Detect credential compromise
- Identify suspicious EC2 activity
- Alert on brute-force and scanning behavior

## SOC Insight
GuardDuty provides alerts; analysts confirm incidents using CloudTrail.
