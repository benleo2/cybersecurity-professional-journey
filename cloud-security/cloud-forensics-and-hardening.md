# Cloud Forensics & Post-Incident Hardening

## Evidence Collection
CloudTrail logs preserved and analyzed for timeline reconstruction.

## Timeline Analysis
Identified initial access, privilege escalation, and malicious actions.

## Root Cause
Credential exposure due to missing MFA and excessive permissions.

## Hardening Actions
- Enforced MFA
- Reduced IAM privileges
- Improved GuardDuty alerting

## Security Outcome
Improved cloud security posture and reduced future risk.
