## What CloudTrail Is
CloudTrail records all AWS account activity including logins and API calls.

## Why CloudTrail Is Critical
It enables detection of unauthorized access, privilege escalation, and account compromise.

## Security Use Cases
- Detect failed and successful logins
- Monitor IAM changes
- Investigate suspicious API calls

## SOC Insight
CloudTrail is the primary forensic and detection log in AWS.
